- `compute_severity(raw_confidence: float, source_count: int = 1, age_days: float = 0) -> SeverityResult`
  - confidence_component = raw_confidence * 10 * FEED_CONFIDENCE_WEIGHT
  - source_component = min(math.log2(source_count + 1) / math.log2(11), 1.0) * 10 * SOURCE_COUNT_WEIGHT
  - recency_component = math.exp(-0.008 * age_days) * 10 * RECENCY_WEIGHT
  - total = round(confidence_component + source_component + recency_component, 2)
  - explanation dict with keys: confidence_component, source_count_component, recency_component, score_version

NOTE: SQLite does not support PostgreSQL's `INSERT ... ON CONFLICT ... RETURNING xmax`. The upsert implementation MUST have a SQLite-compatible fallback for testing. Strategy: use `sqlalchemy.dialects.postgresql.insert` for the PostgreSQL path, but provide a `_upsert_sqlite` fallback that uses SELECT-then-INSERT/UPDATE within the same transaction. The main `upsert_ioc` function detects the dialect from the session's bind and dispatches accordingly.

**backend/app/normalization/upsert.py:**

`async def upsert_ioc(session: AsyncSession, ioc: NormalizedIOC) -> tuple[str, bool]`:
1. Canonicalize the IOC value: `canonical_value = canonicalize_ioc(ioc.value, ioc.ioc_type)`
2. Compute severity: `severity = compute_severity(ioc.raw_confidence, source_count=1, age_days=0)`
3. Check dialect (`session.bind.dialect.name`):
   - If "postgresql": use `pg_insert(IOCModel).values(...).on_conflict_do_update(constraint="uq_iocs_value_type", set_={last_seen: func.now(), source_count: IOCModel.source_count + 1, severity: severity.score, score_version: CURRENT_SCORE_VERSION, score_explanation: severity.explanation, is_active: True, retired_at: None}).returning(IOCModel.id, literal_column("xmax = 0").label("is_new"))`
   - If "sqlite": SELECT existing by (value, type). If exists: UPDATE set last_seen, source_count+1, severity, score_explanation. Return (id, False). If not: INSERT new row. Return (id, True).
4. Always insert an `IOCSourceModel` row with: `ioc_id=ioc_id, feed_name=ioc.feed_name, raw_score=ioc.raw_confidence, raw_payload=ioc.raw_payload, feed_run_id=ioc.feed_run_id`
5. Return `(str(ioc_id), is_new)`

For the severity on update, recompute with the NEW source_count:
- Query current source_count after upsert (or track it from the update)
- `severity = compute_severity(ioc.raw_confidence, source_count=new_source_count, age_days=age_from_first_seen)`

`async def infer_cooccurrence_relationships(session: AsyncSession, ioc_ids: list[str], inferred_by: str, confidence: float = 0.7) -> int`:
1. Import `combinations` from itertools
2. For each (source_id, target_id) pair in `combinations(ioc_ids, 2)`:
   - Check dialect:
     - PostgreSQL: `pg_insert(IOCRelationshipModel).values(source_ioc=source_id, target_ioc=target_id, relationship="observed_with", confidence=confidence, inferred_by=inferred_by).on_conflict_do_nothing(constraint="uq_ioc_relationship")`
     - SQLite: SELECT existing. If not exists, INSERT.
3. Return count of edges inserted (not counting conflicts)
